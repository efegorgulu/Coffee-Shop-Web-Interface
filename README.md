# Coffee Shop Web Interface (HTML & CSS)

A static **coffee shop website interface** built with **HTML5 and CSS3**, featuring a fixed navigation header, hero landing section, product and menu grids, customer reviews, contact form with embedded map, and blog cards styled with a dark modern theme.

## Screenshot
![](Screen.gif)

## Highlights
- **Fixed top navigation bar** with logo, menu links, and interface icons (search, cart, menu)  
- **Hero section** with full-screen background image, overlay gradient, promotional text, and call-to-action button  
- **About section** introducing the coffee brand with image and descriptive content layout  
- **Menu section** displaying multiple coffee items in a responsive grid with pricing and add-to-cart buttons  
- **Products section** showcasing featured coffee products with star ratings and discount pricing  
- **Customer reviews section** with testimonial cards, user images, and rating stars  
- **Contact section** featuring an embedded Google Maps location and styled contact form inputs  
- **Blog section** presenting article cards with hover image zoom effects and post metadata  
- **Responsive layout adjustments** using CSS media queries for smaller screen sizes  

## Technologies Used
- **HTML5**
- **CSS3**
- **Font Awesome** for interface icons
- **Google Maps Embed**

## Installation
1. Clone or download this repository  
2. Open `index.html` in your preferred web browser  

## Project Notes
- Front-end user interface only (**no backend or JavaScript functionality**)  
- Navigation icons and buttons are **visual interface elements only**  
- Contact form is present for layout purposes and does not submit data  
- Images and assets are stored locally inside the `assets/images/` directory  
- Basic **responsive behavior implemented using CSS media queries**
